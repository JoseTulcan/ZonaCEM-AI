📖 User Manual

Complete guide to using ZonaCEM AI

What is ZonaCEM AI?

ZonaCEM AI is a deep-learning application that predicts RF power distribution maps in indoor environments. Given a building layout, antenna position and measurement points, the model generates a complete power map and identifies exposure zones.

Input Layers
🏗️
Structures
Walls, floors and obstacles encoded as grayscale intensity = height (m).
📡
Antenna Position
Single point marking antenna location. Intensity = height (m).
📊
Measurements
Sample power readings (dBm) at known positions. Used for adaptive calibration.
How to Use
  1. Select frequency model
    Go to Models & Eval, choose 1.95 / 2.13 / 2.65 GHz and click Load Model.
  2. Draw your scenario
    In New Scenarios, use the canvas tools to draw structures, place the antenna and add measurement points.
  3. Run prediction
    Click Predict. The backend will run inference and return the power map + zone segmentation.
  4. Analyze results
    Review the predicted power map and the three-zone segmentation (red/yellow/green) with their radii in meters.
